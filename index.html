PocketBuilds

PocketBuilds is plugin ecosystem and set of tools for building custom PocketBase binaries that have been extended with modular Go code.

You can browse for plugins that extend system functionality and create builds + download PocketBase binaries directly from the PocketBuilds website.

PocketBuilds uses an open source project called xpb under the hood to run all the builds. You can use xpb directly in your terminal, and even add your own custom plugins to your projects.
